New in version 3.6:

  • Added the ability to type in text and encode it as a QR Code.
  • Added support for many more locales for web search and product search.
  • Added Hebrew translation.
  • Fixed support for the Motorola Xoom tablet.
  • Provided a choice of creating a new contact or merging with an existing one.
  • Other minor fixes.

New in version 3.53:

  • Added support for 16 and 24 digit ITF codes.
  • Added support for mirrored images for front-facing cameras.
  • Improved Chinese, Turkish, Czech, and German translations.
  • Improved handling of all day calendar events on some devices.

New in version 3.52:

  • Fixed a crash on the Wildfire, Intercept, and other low DPI devices.
  • Improved Russian translation.

New in version 3.51:

  • Improved Data Matrix barcode reading.
  • Added support for Hanzi encoding of Chinese characters.
  • Added a preview of shortened URLs and product barcodes.
  • Added Korean translation.
  • Fixed Danish and Turkish translations.
  • Fixed a couple of rare crashes.
  • Many bug fixes.
